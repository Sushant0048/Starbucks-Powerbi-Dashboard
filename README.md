# ☕ Starbucks Nutritional Analysis Dashboard

![Dashboard Preview](starbucks_dashboard_preview.png)

## 📌 Project Overview
An interactive Power BI dashboard analyzing Starbucks 
beverage nutritional data across 33 beverages and 
9 categories with dynamic filtering capabilities.

## 📊 Dashboard Features
- ✅ 4 KPI Cards (Beverages, Sugar, Calories, Caffeine)
- ✅ Area Chart (Avg Calories by Category)
- ✅ Donut Chart (Category Distribution)
- ✅ Bar Chart (Global Presence)
- ✅ Horizontal Bar (Top Caffeine Beverages)
- ✅ Top 5 Cards with Product Images
- ✅ Protein Range Slider Filter
- ✅ Beverage Prep Dropdown Filter

## 🔍 Key Business Insights
| Metric | Value |
|--------|-------|
| Total Beverages | 33 |
| Average Sugar | 33.02g |
| Average Calories | 194.30 kcal |
| Average Caffeine | 81.35 mg |
| Highest Caffeine | Coffee (293.75mg) |
| Most Popular Category | Classic Espresso (21%) |
